建立目录命令：mkdir -p japan/canglaoshi //在创建连续目录的时候使用-p  单独的目录时候不使用-p

切换目录命令：cd,  cd ~ :进入家目录  cd -:进入上次的目录  cd .. :进入上一级目录  cd .:切换到当前目录
当前目录命令：pwd

删除目录：rmdir [目录名]：删除空目录，不能删除有内容的目录

rm 用来删除目录或者文件的
rm -r japan/  ：-r 表示删除目录，这种情况下删除要询问是否删除
rm -rf japan/ : 强制删除，不需要询问
rm -rf /：这条命令是Linux自杀命令

cp [选项] [原文件或目录] [目标目录]
选项：
-r 复制目录
-p 连带文件属性复制
-d 若源文件是链接文件，则复制链接属性
-a 相当于-pdr

ll命令就是 ls -l


剪切命令： mv  [源文件或目录] [目标目录],不需要加 -r
mv a.txt b.txt ：改文件名
